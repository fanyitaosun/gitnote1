1 搭建

之前搭建过node 卸载之后安装cnpm 无法安装 安装其它的软件 也无法安装

修改node的

查询当前配置

npm config ls

更换存储配置

npm config set prefix"D:\nodejs\node_modules\npm\node_global_modules"

npm config set cache"D:\nodejs\nodemodules\npm\nodecache"

npm config set registry https://registry.npm.taobao.org


2.升级npm 增加可视化 vue-ui

先卸载 旧版本 在安装新版本

npm uninstall -g vue-cli

npm install -g @vue/cli


问题：vue ui很好用 但是如果不小心手动创建项目的时候 搭建脚手架的时候 使用旧版本的命令 你就切换到 旧版本2.9 这个时候需要把就版本的cli 卸载 重新安装。
